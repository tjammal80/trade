# Hub Trend Gold v3.3

هذه النسخة تركز على استقرار مسار التقارير قبل الانتقال إلى الاشتراكات والدفع.

## ما الجديد
- حفظ كمسودة / نشر مباشر من صفحة التحرير
- معاينة داخلية قبل النشر
- نسخ احتياطي تلقائي قبل أي استبدال أو حذف أو استعادة
- صفحة لإدارة النسخ الاحتياطية واستعادة أي نسخة سابقة
- تحقق أكثر صرامة لبنية JSON قبل الحفظ
- حماية أفضل لعرض المسودات: المسودات لا تظهر للعامة
- API health أوسع لفحص البيئة والنسخ الاحتياطية

## المسارات المهمة
- `/admin/index.php`
- `/admin/report-edit.php`
- `/admin/report-import.php`
- `/admin/report-template.php`
- `/admin/report-history.php?slug=...`
- `/api/health.php`
- `/api/reports.php`

## بيانات الدخول الافتراضية
- Admin: `admin / ChangeThisAdminPass!`
- Member: `member@example.com / ChangeThisMemberPass!`

غيّر هذه القيم فورًا من:
`src/Support/config.php`

## ملاحظات مهمة
- لا ترفع الحزمة فوق ملفات التقارير الحية قبل أخذ نسخة احتياطية محلية.
- النسخ الاحتياطية تحفظ داخل `storage/backups/{slug}/`.
- إذا أردت منع المستخدمين من رؤية أي تقرير إلا المنشور، اترك الحالة Draft حتى تراجع المعاينة.


## v3.3.1
- تم فصل التحقق بين المسودة والنشر.
- المسودات تُحفظ بحد أدنى من الحقول، بينما النشر يفرض التحقق الكامل.
- لا تحتوي هذه الحزمة على ملفات تقارير جاهزة داخل storage/reports حتى لا تستبدل تقاريرك الحالية.
